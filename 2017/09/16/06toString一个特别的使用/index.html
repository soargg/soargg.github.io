toString一个特别的使用

​ 之前在网上看到过一道不算多么变态的面试题：定义一个函数，实现add(1)(2)(3)(4)(5)…，最后返回一个所有参数的累加。

toString()函数

​ 该函数返回值为String类型，返回当前对象的字符串形式；所有的对象内部或者是通过原型继承都具有toString()方法。

类型 行为描述
Array 将 Array的每个元素转换为字符串，并将它们依次连接起来，两个元素之间用英文逗号作为分隔符进行拼接。（对象实例的行为）
Boolean 如果布尔值是true，则返回”true”。否则返回”false”。
Date 返回日期的文本表示。（对象实例的行为）
Error 返回一个包含相关错误信息的字符串。（对象实例的行为）
Function 返回如下格式的字符串，其中 functionname 是一个函数的名称，此函数的 toString 方法被调用： “function functionname() { [native code] }”
Number 返回数值的字符串表示。还可返回以指定进制表示的字符串，请参考Number.toString()。
String 返回 String 对象的值。
Object(默认) 返回”[object ObjectName]”，其中 ObjectName 是对象类型的名称。
  • Array，Date等，属于js内置对象，直接调用toString(),因为是构造函数，返回结果属于Function调用toStrion()行为。
